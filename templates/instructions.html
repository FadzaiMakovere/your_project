Instructions for Filling Out the Form

General Instructions

  • Ensure all information is accurate before submitting.
  • Use numbers only where numerical input is required.
  • Do not include units (e.g., type 120, not 120 mmHg).
  • Leave fields blank only if marked as optional.

Before You Submit

  • Review all entered values carefully.
  • Confirm all required fields are filled in.
  • Click the “Submit” button once you're ready.
  • Wait for the confirmation message to ensure successful submission.
